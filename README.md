# Contextly 🤖

**Contextly** is an AI-powered, Slack-integrated assistant that eliminates team misalignment and tool-switching. By passively listening to relevant Slack channels and integrating with your team's ecosystem (starting with Jira), Contextly surfaces exactly what’s going on, who’s involved, and where things stand—right where your team already works.

### 🚀 Key Features
* **Passive Channel Listening:** Autonomously tracks context within Slack threads and channels.
* **Jira Integration (MVP):** Connects tickets and status updates directly to ongoing conversations.
* **Future Roadmap:** Expanding to Notion docs, Confluence pages, and Google Drive.
